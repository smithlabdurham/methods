Loading the data

elec <- read.csv("data/uk_electricity.csv")
head(elec)
  • What are the columns?
  • What years does it cover?
  • What units are used?
  • [10 min] Live walkthrough. Load the data, show head(), str(), nrow(). Narrate: “This is like pd.read_csv() in Python. head() is like df.head().”
  • Point out: columns are fuel types, values are TWh.

Your first plot

plot(elec$year, elec$biomass_twh,
     xlab = "Year", ylab = "TWh",
     main = "UK biomass electricity generation")

“What’s the first thing you notice?”

  • The trend is steeply upward. Students should notice the rapid growth.
  • Ask: “How does this compare to what you expected?”
  • If someone asks about the scale — good. “Is that a big number?” comes next week, but plant the seed now.

Independent exercises

Work through the exercises

  2. Add coal to the same plot — what do you see?
  3. Calculate mean and SD of biomass for the last 5 years

Git

📋🎓💻 Setting up your repo

Why Git?

Your code needs to be:

  • Saved — not just in the browser
  • Versioned — so you can go back if something breaks
  • Visible — so your demonstrator can see your progress

Git does all three. GitHub Desktop makes it painless.

  • [5 min] Brief motivation. Don’t oversell Git — just frame it as “how professionals save their work.”

The workflow

Every session:

  1. Open GitHub Desktop
  2. Pull any changes
  3. Do your work in .R files
  4. Commit with a message
  5. Push

That’s it.

Two clicks and a sentence.

Your commit history becomes your learning record.

  • Emphasise simplicity. Resist the urge to explain branches, merges, or anything advanced. Week 5 will cover collaboration.

Live demo

🖥️ Switching to GitHub Desktop

  1. Clone your personal repo (from GitHub Classroom link)
  2. Open the folder — see the starter files
  3. Create week1.R — paste your WebR code
  4. In GitHub Desktop: see the change → commit → push

  • Show GitHub.com afterwards: “See? Your code is now on the internet. Your demonstrator can see it. You can access it from any computer.”

Discussion

What did the data show?

“What did you find in the data?”

  • Biomass has grown rapidly
  • Coal has collapsed
  • Biomass is now a significant fraction of UK electricity
  • [15 min] Pull the room together. Collect observations.
  • Ask: “If biomass is counted as zero-carbon, what happens to the UK’s official emissions figures?” Let students reason through this.
  • Don’t give the answer. They’ll dig into it in Week 2.

“If biomass is counted as zero-carbon, what happens to the UK’s official emissions figures?”
